.. _launch:

Launch Application
==================

You can either execute opGUI after the installation procedure is completed, follow the next steps:

.. tabs::
   .. tab:: Windows 11
  
      Go to the project diretory which shall be placed on your home
      
      .. code-block:: 

         cd %USERPROFILE%\opgui

      Launch the application
  
      .. code-block:: 

         yarn win-launch
  
   .. tab:: Debian
  
      Go to the project diretory which shall be placed on your home
  
      .. code-block:: 

         cd $HOME/opgui

      Launch the application
  
      .. code-block:: 

         yarn deb-launch
